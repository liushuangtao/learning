# learning
个人后端技术学习沉淀仓库，持续更新日常练习代码、学习笔记与实战案例，纯开源非商业项目。

## 项目介绍
本仓库用于日常编程技术学习、代码实操练习、开发经验整理归档，主要记录 Java 后端开发、服务端运维、中间件使用等相关学习内容，方便自我复盘总结，也可供技术同行参考交流。

## 技术栈
- 编程语言：Java
- 主流框架：SpringBoot、SpringCloud、MyBatis
- 数据库：MySQL、SQL 优化实战
- 中间件：Redis 基础使用与实操
- 运维技术：Linux 常用操作、服务器部署相关笔记
- 开发工具：Maven、Git、Docker 基础实践
- 其他：算法练习、开发踩坑总结、通用工具类封装
